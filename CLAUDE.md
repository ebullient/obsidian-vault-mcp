# Obsidian Vault MCP

Obsidian plugin exposing the vault as an MCP server over HTTP. See [README.md](README.md) and [CONTRIBUTING.md](CONTRIBUTING.md).

## Code style

- **Line length**: 80 characters
- **Method chaining**: Break at dots, even for single chains
- **Naming**: `vaultasmcp-` prefix for source files
- **UI text**: Sentence case; complete sentences ending with a period; semicolons between related clauses; no parenthetical notes.

## QA

- `npm run build` — lint + build
- `npm run lint` — lint only
- `npm run fix` — auto-fix lint issues
- Reference issues by `file.ts:123`
